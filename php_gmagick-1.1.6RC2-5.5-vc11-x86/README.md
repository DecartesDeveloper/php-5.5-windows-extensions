1) Extract GraphicsMagick-1.4.02-Q16.7z to C:\GraphicsMagick-1.4.02-Q16
2) Add C:\GraphicsMagick-1.4.02-Q16 to Path environment variable
3) Set new environment variable MAGICK_HOME to C:\GraphicsMagick-1.4.02-Q16
4) Enable gmagick extension by editing your php.ini by appending extension=php_gmagick.dll
5) Restart
6) Profit

Of course it is possible to install GraphicsMagick to a directory of your choice, just update the environment variables accordingly and restart.
